# Naveen Muralikrishnan — Terminal Portfolio

A terminal-style personal portfolio website inspired by command-line workflows.

## Live Features

- Terminal-like boot/loading sequence
- Interactive command input with history and autocomplete
- Portfolio sections rendered via commands
- Responsive UI for desktop and mobile
- Custom theme using personal color palette

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## Available Commands

- `help` — list all available commands
- `ls` — list virtual files
- `cat [file]` — read file-style portfolio sections
- `pwd` — print current path
- `whoami` — identity summary
- `neofetch` — quick profile/system-style card
- `about` — introduction
- `skills` — technical stack
- `experience` — work experience
- `education` — education details
- `contact` — contact info
- `hobbies` — hobbies
- `links` — external links
- `open [link]` — open link in new tab
- `history` — command history
- `date` — current date/time
- `echo [text]` — print text
- `clear` — clear terminal output
- `surprise` — random fun output

## Project Structure

- `index.html` — app shell and layout
- `styles.css` — theme and responsive styles
- `loading.js` — boot screen animation logic
- `script.js` — terminal command engine
- `commands/` — content fragments used by command outputs
